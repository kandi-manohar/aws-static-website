üìå What is a Static Website?

A static website consists of fixed content such as HTML, CSS, and JavaScript files. It does not require a backend server or database. The same content is delivered to every user, making it fast and reliable.

‚òÅÔ∏è How AWS S3 Static Website Hosting Works

Amazon S3 (Simple Storage Service) allows you to store files in buckets. When static website hosting is enabled:

  • HTML and CSS files are stored as objects inside an S3 bucket
  • S3 acts as a web server and serves files over HTTP
  • An index document (index.html) is returned by default
  • A bucket policy allows public read access

‚öôÔ∏è Why Use Amazon S3?

  • Highly scalable and reliable
  • Very low hosting cost
  • No server management required
  • Integrates with CloudFront for CDN
  • Easy deployment and maintenance

üîê Security & Access

Access to the website is controlled using an S3 bucket policy that allows public read-only access. Write or delete operations are restricted to the bucket owner.
